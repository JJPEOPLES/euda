## ğŸš€ Getting Started

### Online Version
Visit [euda.app](https://euda.app) to start using Euda instantly in your browser.

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/euda-app/euda.git
   cd euda
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## âŒ¨ï¸ Keyboard Shortcuts

### Global
- `Ctrl/Cmd + N` - New file
- `Ctrl/Cmd + O` - Open file
- `Ctrl/Cmd + S` - Save file
- `Ctrl/Cmd + Shift + P` - Command palette

### Text Editor
- `Ctrl/Cmd + F` - Find
- `Ctrl/Cmd + H` - Find and replace
- `Ctrl/Cmd + /` - Toggle comment
- `Alt + Up/Down` - Move line up/down

### Editor Switching
- `Ctrl/Cmd + 1` - Text Editor
- `Ctrl/Cmd + 2` - Image Editor
- `Ctrl/Cmd + 3` - Video Editor
- `Ctrl/Cmd + 4` - Audio Editor
- `Ctrl/Cmd + 5` - Document Editor
